# LCOLONQ MICROGAME JAM 2K26 GODOT TEMPLATE

## Test for Web
Select "Remote Display", the 4th icon at the top-right of the Godot Editor

Select "Run in Browser" to test the game with a custom harness capable of setting
the difficulty within your custom range before playing your game

Select "Re-export Project" to quickly update your export, allowing you to refresh the 
browser to see your changes. (CTRL + R will not work because the game will intercept it)

## Final Export:
Project > Export... > Export All... > Release

This should populate the output/ with your game

Zip the output/ folder for shipping
